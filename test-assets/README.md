This assets are prepared for testing Tokenizers.DotNet.
After adding a new tokenizer files, please update following code files to use it:

- `dotnet/Tokenizers.DotNet.Test/Internals/ModelId.cs`
- `dotnet/Tokenizers.DotNet.Test/Internals/Models.cs`

Currently tokenizer which filename is `tokenizer.json` is supported. Check `Models.cs` for details.
